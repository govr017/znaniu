Основы
HTML - это язык программирования сделанный для маркирования документов и веб страниц.
Работать в нём довольно просто. Всё в HTML состоит из тегов:
<b>hello</b> - текст будет выглядеть жирным
<p> - это открывающий тег
</p> - закрывающий
всё посередине получит функцию тега

https://html5book.ru/html-tags/ - список тегов HTML
