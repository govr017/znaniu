Основы
CSS - это язык описания внешнего вида документа. В CSS можно изменить по сути всё что угодно о том как что то выглядит на вебсайте. В качестве первого урока мы разберём синтаксис(как всё пишется). Синтаксис CSS выглядит примерно вот так:
.maintxt {
margin-top: 5px;
margin-left: 220px;
font-family: 'Source Code Pro', monospace;
font-size: large;
color:greenyellow;
background-color:#201e21;
border-radius: 5px;
}
body {
background-color:#312f33;
}

Как можно заметить код в CSS начинается с декларации что менять. Если перед названием стоит точка то тогда это класс и его можно будет использовать на чём угодно. Если хочется чтобы каждый определённый тег HTML имел одни и те же свойства(например у всех тегов p чтобы былы большие буквы) то тогда точку писать не надо и надо написать название тега прям как в примере 2.
Чтобы использовать нами написанный код в CSS то можно добавить его к уже существующему тегу(добавив параметр class="вашкласс") но если текст без тега или нужно добавить второй класс то можно воспользоваться тегом div
